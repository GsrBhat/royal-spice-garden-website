# Royal Spice Garden ─ Premium Restaurant Website Concept

A state-of-the-art, premium, and fully responsive concept landing page designed for **"Royal Spice Garden"**—a high-end fine dining and Indian-fusion restaurant. Built with rich obsidian and warm champagne-gold aesthetics, this project showcases modern front-end capabilities, custom visual assets, interactive UI/UX features, and automated reservation management.

---

## 💼 Case Study (For Fiverr, LinkedIn, and Portfolio)

### 🔴 The Problem
Luxury fine-dining restaurants rely heavily on visual prestige and immediate brand authority to attract patrons. A generic, slow, or poorly structured web presence fails to communicate the high culinary standard and unique ambiance of the dining experience, resulting in missed booking opportunities and lower customer acquisition.

### 🟢 The Solution
Built **Royal Spice Garden**—a high-conversion, responsive web experience featuring:
*   A premium visual story and Chef's narrative section to build brand authority.
*   An interactive menu catalog categorized by courses to keep users engaged.
*   An elegant, real-time client-side validated reservation booking card.
*   A custom vector-based location navigation map card matching the dark-luxury theme.
*   A fully responsive layout matching mobile-first specifications.

### 🔵 The Result
*   **Elevated Brand Prestige:** An elite UI design that matches the standard of venture-backed SaaS startups and Michelin-starred dining.
*   **Frictionless Booking Experience:** Easy-to-use booking layout that reduces steps to complete reservations, increasing online reservation conversions.
*   **High Engagement:** Interactive elements (tab switches, lightbox modal, testimonial sliders) keep prospective diners engaged on the site longer.

---

## ⚡ Performance Metrics & Key Features

*   **📱 Mobile Friendly & Mobile-First Optimization:** Fluid media-query structures that scale down cleanly from ultra-wide 4K monitors to compact iPhone screens.
*   **⚡ Modern UI/UX & Glassmorphism:** Velvet-backdrop color scales (`#070605` obsidian & `#110F0D` truffle), champagne-gold accent variables, floating animated showcase elements, and micro-hover states.
*   **✨ Responsive Design:** Flexible flexbox and grid layouts providing 100% responsiveness without any overlapping contents.
*   **🔍 SEO Optimized:** Structured markup with semantic HTML5 tags (`<header>`, `<section>`, `<footer>`), descriptive alt-attributes, unique interactive element IDs, meta descriptions, and custom head typography.
*   **📅 Online Reservation System:** Client-side validated booking form that locks out past dates, verifies input criteria in real-time, and presents a customized animated success screen.
*   **🍽️ Interactive Menu Categories:** Tabbed selection panel that lets users switch between appetizers, main courses, desserts, and signature cocktails instantly with smooth fade-in animations.
*   **🖼️ Visual Lightbox Gallery:** Masonry-style image grid featuring click-to-zoom modals, next/previous arrow navigation, ESC-key dismissals, and hover-triggered details overlays.

---

## 🛠️ Technology Stack
*   **Core Structure:** Semantic HTML5
*   **Styling System:** Vanilla CSS3 (Custom variables, flex-grids, keyframe animations, glassmorphism)
*   **Interactions:** Vanilla JavaScript (ES6+, IntersectionObserver, client-side validation, sliders, modal controls)
*   **Icons:** Lucide Icons (CDN)
*   **Fonts:** Google Fonts (`Cormorant Garamond` serif and `Plus Jakarta Sans` sans-serif)
*   **Bundler:** Vite (for fast local development and assets bundling)

## 🚀 Running Locally

1.  Navigate to the project directory:
    ```bash
    cd "demo web"
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Build for production:
    ```bash
    npm run build
    ```
